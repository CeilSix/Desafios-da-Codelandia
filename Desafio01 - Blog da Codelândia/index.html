Descoberta comunidade de hackers formada por adolescentes

A maioria dos membros são menores, entre 11 e 16 anos. Entre suas conversas estão como utilizar meios de pagamento dos pais sem eles saberem, troca de gift cards para compra de malware e como hackear os computadores de professores. Segundo o pesquisador de segurança Jan Holman, esses tipos de comunidades são atraentes para jovens, já que o hacking é visto como algo “divertido”. Além disso, o local também oferece uma chance para aprender programação.

Tim Berners-Lee é cético em relação à Web3

Apesar de o criador da web acreditar que é preciso criar uma nova arquitetura descentralizada, devolvendo aos usuários o controle de seus dados, uma tecnologia baseada em blockchain “simplesmente não funciona”. Berners-Lee propõe uma nova plataforma chamada Solid, construída utilizando ferramentas web padrão e especificações abertas.
